Global Toggles

When per-site rules are off, these apply on every site. When per-site rules are on for shopping or Pinterest, only sites you did not add to the list still follow these global switches.

Per-Site Overrides

This is not an “only show here” list. Each site you add gets its own allow/block switch.

  • Allow (checkbox on) = Linger may show shopping nudges or Pinterest prompts on that site (other settings like snooze still apply).
  • Block (checkbox off) = Linger will not pop up on that site.
  • Sites you never add to the list still follow the global toggles above, not the per-row checkboxes.

Use hostnames only (e.g. amazon.com), without https://. Turn on per-site mode below, then add sites.
